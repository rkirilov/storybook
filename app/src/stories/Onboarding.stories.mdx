# Set Up the React Project

The easiest way to start with React is to use <a target="_blank" href="https://github.com/facebook/create-react-app">create-react-app</a>. To scaffold your project structure, follow the <a href="https://github.com/facebook/create-react-app#creating-an-app">installation instructions</a>.

```shell
npx create-react-app my-app --use-npm
cd my-app
npm start
```
or
```shell
yarn create react-app my-app
cd my-app
yarn start
```

After creating react app you need to add support for sass

- Install node-sass:
```shell
# using npm
npm install node-sass --save

# using yarn
yarn add node-sass
```
- Convert your <code>.css</code> to <code>.scss</code>.
- Import your <code>.scss</code> files in your React components like <code>App.js</code>

```js
import './App.scss';
```
> More information about using sass stylesheet in create react app you can find in <a target="_blank" href="https://create-react-app.dev/docs/adding-a-sass-stylesheet/">Create React App documentation</a>.

Before you start playing with < target="_blank" href="https://www.telerik.com/kendo-react-ui/components/introduction/">KendoReact components</>, let's clean up the sample app a bit. Here is a list of suggested edits, with the resulting <code>src/App.js</code> code below:

- src/App.js
  - Remove everything inside the App container div: <code>`<div className="App"> ... </div>`</code>
  - Remove the logo.svg import
  - Import <code>{Component}</code> in addition to React
- Remove the contents of <code>src/App.scss</code>
- Delete <code>src/logo.svg</code>

```js
import React, {Component} from 'react';
import './App.scss';

class App extends Component {

  render() {
    return (
      <div className="App">
        <h1>Hello KendoReact!</h1>
      </div>
    );
  }
}

export default App;
```

# Install KendoReact components

KendoReact is distributed as multiple NPM packages, scoped to <code>@progress</code>. For example, the name of the Button package is @progress/kendo-react-buttons. KendoReact is a rich suite of <a target="_blank" href="https://www.telerik.com/kendo-react-ui/components/">many modular components</a>.
Let's add button component package and its dependencies

```shell
npm install --save @progress/kendo-react-buttons @progress/kendo-licensing
```

# Import KendoReact components

Once installed, import the package module.

```js
// ES2015 module syntax
import { Button } from '@progress/kendo-react-buttons';
```
```js
// CommonJS format
const { Button } = require('@progress/kendo-react-buttons');
```
