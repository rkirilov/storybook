# Outline Button

The KendoReact Button provides a clickable UI functionality and enables you to display only textual content, or show predefined icons, images, and custom icons, and to render a combination of textual and image content.

>If you getting started with <a target="_blank" href="https://www.telerik.com/kendo-react-ui/components/introduction/">KendoReact components</a> you may need to create a react app. Please read the <LinkTo story="page" kind="getting-started-onboarding">Setup the React Project</LinkTo> section in the getting started article.

## Installation

1. Download and install the package. Use Node.js v5.0.0 or later.

```shell
npm install --save @progress/kendo-react-buttons @progress/kendo-licensing

```

2. Once installed, import the package module.

```js
// ES2015 module syntax
import { Button } from '@progress/kendo-react-buttons';
```
```js
// CommonJS format
const { Button } = require('@progress/kendo-react-buttons');
```
3. Install your custom theme.

  3.1. You can download your design system from here: <a target="_blank" href="design-system/test-design-system.zip" download="test-design-system.zip">Test Design System</a>. The download is a zip file which contains Test Design System package.

  3.2. Once you download it unzip it in the root folder of your application

  3.3. Navigate to the design system folder and install npm modules for your design system


```shell
cd test-design-system
npm install
```
  3.4. Add the test-design-system package in your application <code>package.json</code> file

```js
  "dependencies": {
    ...
    "test-design-system": "file:./test-design-system"
  },
```
> <code>file:./test-design-system</code> value is the relative path to the <code>test-design-system</code> folder. For example if you put it next to your application folder the value should be <code>file:../test-design-system</code>

4. Import theme package in your application in the <code>App.scss</code> file
```js
@import '~test-design-system/src/styles';
```

5. Install design system package in your project
```shell
cd ..
npm install
```

6. Follow the instructions on the <a target="_blank" href="https://www.telerik.com/kendo-react-ui/my-license/">KendoReact My License page</a>to activate your license. You can skip this step if your application already contains a KendoReact license file.
